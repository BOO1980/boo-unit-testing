Unit Testing Overview

  • Unit testing is a type of software testing where individual units or components of a software are tested
  • The purpose of the unit testing is to validate that each unit of the software code performs as expecvted
  • unit testing is done during the developent of an application by developers
  • Unit test isolate a section of code and verify its correctness
  • a unit may be an individual function, method, procedure, module or object
  • During the system development life cycle unit testing is first level of testing done before integration testing
  • unit testing is white box testing technique that is usually performed by the developer
  • QA engineers can also do unit testing
